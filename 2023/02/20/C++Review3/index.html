对于const的顶层底层再理解

区分const修饰时，以指针符星号为界，往左侧进行修饰。const修饰类型时，则为底层const，表示指针指向一个常量类型对象，如果是顶层const，表示修饰的是指针为常量，一般在指针的右侧。
例如： const char与char const 是一样的，这里的const都是修饰类型，而char * const则是顶层const，表示修饰的是指针为常量指针。
