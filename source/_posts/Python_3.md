# Python Note 3

+ ### Classes
+ Use Pacal naming convertion,not like variables/function use lower cases and underscore connection.
+ Basic usage:
```
class Point:
	def draw(self):
		print("draw")

	def print(self):
		print("this is point")


point1 = Point()
point1.x = 5
point1.draw()
print(point1.x)
```
